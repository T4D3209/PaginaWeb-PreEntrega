Este proyecto consiste en el desarrollo de una página web tipo ecommerce orientada a la venta de periféricos para PC.
La plataforma permitirá a los usuarios explorar distintos productos tecnológicos, visualizar información detallada, contactarse con el soporte por medio de un formulario y realizar compras de manera simple e intuitiva.

El objetivo principal es crear una interfaz moderna, responsive y fácil de usar para mejorar la experiencia del usuario al momento de buscar productos relacionados con gaming, oficina y computación en general.
